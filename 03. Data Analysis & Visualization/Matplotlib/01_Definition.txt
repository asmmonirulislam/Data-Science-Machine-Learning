Matplotlib is a Python library used mainly for data visualization—turning data into graphs, 
charts, and plots so you can understand patterns easily.

Here are its main uses:

1. Data visualization (core use)

Matplotlib helps you convert raw data into visual forms like:

Line plots (trends over time)
Bar charts (comparisons)
Scatter plots (relationships between variables)
Histograms (data distribution)
Pie charts (proportions)
2. Exploratory Data Analysis (EDA)

In data science, before building models, you use matplotlib to:

Spot trends and patterns
Detect outliers
Understand data distribution
Compare variables visually

Example: seeing how salary changes with experience.

3. Machine Learning visualization

Used to:

Plot training vs validation accuracy
Visualize loss curves
Show confusion matrices
Compare model performance
4. Custom plotting control

Matplotlib gives full control over:

Titles, labels, legends
Colors, styles, grid lines
Figure size and layout

So you can design highly customized professional charts.
